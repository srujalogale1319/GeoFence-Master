# GeoFence Master üåç

GeoFence Master is a browser-based geofencing web application that allows users to draw custom geofences on a map and receive real-time entry/exit notifications for themselves and connected devices.

This project is built entirely using frontend technologies and runs without any backend server.

---

## üöÄ Features

- Interactive map using OpenStreetMap
- Draw geofences (Polygon, Circle, Rectangle)
- Real-time geofence entry/exit detection
- Device connection simulation
- Live location tracking using browser Geolocation API
- Notifications panel for all events
- User authentication using LocalStorage (Login & Register)
- Responsive UI with Tailwind CSS & Bootstrap

---

## üõ† Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Leaflet.js (Maps)
- Leaflet.draw (Drawing tools)
- Turf.js (Geospatial calculations)
- Tailwind CSS (UI styling)
- Bootstrap 5 (Auth pages)
- Browser LocalStorage (Auth & data persistence)
